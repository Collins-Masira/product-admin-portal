# Product Admin Portal (React SPA)

## Description
A React-based admin dashboard for managing products using full CRUD operations.

## Features
- Add products (Create)
- View products (Read)
- Edit products (Update)
- Delete products
- React Router navigation
- Custom hooks
- JSON server backend

## Tech Stack
- React
- React Router
- JSON Server
- CSS
- Vitest

## How to Run

### Frontend
npm install
npm run dev

### Backend
npx json-server db.json --port 5000
